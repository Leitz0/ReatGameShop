Para iniciar a api:

1. Instale as dependências:
```bash
cd ./api
npm install
```

2. Inicie o servidor:
```bash
npx json-server db.json -p '3001'
```

3. Acesse a api em:
```bash
http://localhost:3001
```

4. Rotas disponíveis:
```bash
GET /users
GET /users/:id
POST /users
GET /produto
GET /produto/:id
GET /pedido
GET /pedido/:id
POST /pedido
```

Para iniciar o Vite:

1. Instale as dependências:
```bash
cd ./reactGame
npm install
```

2. Inicie o servidor:
```bash
npm run dev
```

3. Acesse a aplicação em:
```bash
http://localhost:5173
```
